First-time setup

1. Go to github.com/settings/tokens?type=beta and create a fine-grained Personal Access Token. Give it permission to Contents: write AND Actions: write on this repo only. Set an expiration (a year is fine).

2. Copy the token (starts with github_pat_) and paste it above. It's stored only in your browser, never sent anywhere except GitHub.

3. Bookmark this page. From now on, just open it, tick the shoots you want, and click Save.

The same token also works for refresh.html.
